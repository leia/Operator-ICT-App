# Operator-ICT-App

Zadání pro .NET vývojáře 
Vytvořte mikroslužbu v ASP.NET Core (třeba spravující osoby), která: 

• Využívá REST 

• Používá JSON 

• Má základní CRUD metody 

• Podporuje Swagger. 

• Je zabezpečená, používá JWT 

• Má volně dostupnou metodu pro přihlášení (vrací JWT) 

• Ukládá data v MSSQL, používá EF code first, používá migrace, používá repository pattern
